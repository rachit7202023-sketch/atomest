# OneClick — The Internet's Toolbox

You are an expert full-stack software engineer, UI/UX designer, SEO specialist, and performance engineer.

Your task is to build a production-ready web application called **OneClick**.

## Vision

OneClick is the internet's toolbox.

It is a place where anyone can instantly access useful online tools without creating an account or paying.

The experience should be extremely fast, clean, minimal, and professional.

Design inspiration:
- Apple
- Vercel
- Linear
- Notion
- Stripe

The website should feel modern, premium, and trustworthy.

---

## Tech Stack

Use:

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Lucide Icons

The codebase must be clean, modular, reusable, and scalable.
